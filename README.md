# dwg sheet renamer

dwg sheet renamer is a python desktop utility for previewing engineering drawing filename conversions.

version 1 is planned as a read-only prototype. it will scan a selected folder, identify likely dwg sheet files, and show proposed filename conversions before any rename feature is added.

no filesystem changes are implemented in this initial project skeleton.

## planned version 1 scope

- select one folder
- scan only the selected folder
- ignore subfolders
- detect `.dwg` files
- identify likely engineering sheet drawings
- ignore known non-sheet drawings
- preview decimal and hundreds-style filename conversions
